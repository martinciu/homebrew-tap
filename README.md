# Homebrew tap

Homebrew formulas for [martinciu](https://github.com/martinciu)'s tools.

## Usage

```sh
brew install martinciu/tap/<formula>
```

## Available formulas

- [ccpulse](https://github.com/martinciu/ccpulse) — Native Go TUI dashboard for Claude Code usage

## Notes

Formulas in this tap are published automatically by [GoReleaser](https://goreleaser.com/) when the source repo cuts a release. Don't edit `Formula/*.rb` by hand — your changes will be overwritten on the next release.
